# 📊 Python Selenium Web Scraper

A Python-based web scraping tool that uses Selenium to continuously extract data from a website and store it in Excel files. The scraper runs in real-time until manually stopped, while ensuring duplicate data is ignored and outputs remain clean and structured.

---

## 🚀 Features

- 🔄 Continuous data scraping (runs until stopped)
- 📁 Automatically generates Excel files with timestamps
- ❌ Prevents duplicate data entries
- ⚡ Automated browser interaction using Selenium
- 📊 Structured data storage for easy analysis

---

## 🛠️ Tech Stack

- Python  
- Selenium  
- Pandas / OpenPyXL
